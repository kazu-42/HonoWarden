# Code of Conduct

HonoWarden follows the Contributor Covenant Code of Conduct.

## Expected Behavior

- Be respectful and constructive.
- Keep technical disagreement focused on evidence, risk, and project goals.
- Avoid harassment, personal attacks, or exclusionary language.
- Respect maintainers' decisions about scope, security posture, and review requirements.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior harms the project or its contributors.

If you need to report conduct concerns, open a minimal issue asking for a private maintainer contact without including sensitive personal details.
