# Packet 04: Verification

## Objective

Validate evidence quality locally and through GitHub before merging.

## Do

- Run formatting checks, brand scan, `git diff --check`, workflow verifier, and
  ops readiness packet.
- Run local review before PR merge.
- Push a branch, open PR, wait for CI, and merge with admin if local review and
  CI are clean.

## Do Not

- Use Spark for QA.
- Skip readback after merge.
